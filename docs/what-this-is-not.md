# What This Is Not

This repo uses autoresearch language carefully. The goal is to keep the artifact useful without making claims it cannot support.

## Not An Autonomous Scientist

This framework does not conduct science on its own. It does not generate validated discoveries. It does not replace domain experts, reviewers, or research teams.

It provides a structure for organizing disagreement and deciding what to inspect next.

## Not A Systematic Review Replacement

This framework borrows discipline from evidence synthesis, but it is not a substitute for a systematic review, scoping review, meta-analysis, rapid review, or living review.

It does not provide a complete protocol for database search, inclusion criteria, duplicate screening, risk-of-bias assessment, reporting standards, or meta-analysis.

When a formal systematic review is required, use the relevant review method and reporting standard.

## Not A Truth Machine

The loop can make disagreement clearer. It cannot make contested evidence uncontested.

A clean template does not make weak sources strong. A rubric score does not make a theory true. A high-volume citation cluster does not prove independent support.

## Not Biomedical Advice

Do not use this repo to make biomedical, clinical, legal, safety-critical, or public-health decisions without qualified expert review and domain-specific methods.

The included example is fictional and non-biomedical on purpose.

## Not A Benchmarked Scientific-Discovery Engine

This repo does not claim benchmarked discovery performance. It does not include an automated experiment runner, model evaluator, theorem prover, simulator, or objective-function optimizer.

It is designed for domains where those signals are unavailable, ambiguous, or insufficient.

## Not A Confidence Generator

The framework should often reduce confidence in broad claims. That is expected.

If a first pass produces a clean winner, no unresolved conflicts, and no undercutting evidence, treat that as a reason to inspect the process.

## Not A Way To Launder AI Output

AI can help draft cards, propose conflicts, summarize sources, or suggest search terms. Those outputs still require human review.

Source artifacts remain more authoritative than generated synthesis.
