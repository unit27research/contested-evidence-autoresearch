# Adjacent Work And Novelty Boundary

This project sits near several established fields. The useful public claim is not that these methods are new. The useful claim is that they can be arranged into a practical anti-convergence loop for unresolved domains.

## AutoResearch And AI Scientist Systems

Autoresearch systems use iterative loops to generate, test, and improve research ideas. Recent examples include systems that write code, run experiments, evaluate results, and author papers.

Closest overlap:

- iterative research loops
- search over possible research moves
- process improvement over repeated cycles
- automated or semi-automated review

Key difference:

- this framework is for domains without a clean objective function
- the loop does not optimize a benchmark, loss curve, simulation, or executable experiment
- the target is disagreement quality and next-question value

Representative references:

- Lu et al., 2024. "The AI Scientist: Towards Fully Automated Open-Ended Scientific Discovery." arXiv:2408.06292. https://arxiv.org/abs/2408.06292
- Yamada et al., 2025. "The AI Scientist-v2: Workshop-Level Automated Scientific Discovery via Agentic Tree Search." arXiv:2504.08066. https://arxiv.org/abs/2504.08066

## Metacognitive Co-Regulation

Agent loops can fixate. Metacognitive and co-regulatory designs try to monitor the reasoning process, not just the task output.

Closest overlap:

- anti-fixation
- explicit process monitoring
- role separation between generator and reviewer
- critique of single-track agent loops

Key difference:

- this project applies the concern to evidence synthesis and contested theory evaluation
- the co-regulation is embedded in artifacts and review roles, not only agent architecture

Representative reference:

- Xu, Martelaro, and McComb, 2026. "Supervising Ralph Wiggum: Exploring a Metacognitive Co-Regulation Agentic AI Loop for Engineering Design." arXiv:2603.24768. https://arxiv.org/abs/2603.24768

## Systematic Review Automation

Systematic review automation studies how AI can help with search, screening, extraction, appraisal, and synthesis.

Closest overlap:

- source discipline
- extraction templates
- evidence appraisal
- provenance
- human review

Key difference:

- this repo is not a systematic review protocol
- it does not claim comprehensive search or formal risk-of-bias coverage
- it is aimed at early structured reasoning in contested fields, not formal evidence synthesis

Representative reference:

- de la Torre-Lopez, Ramirez, and Romero, 2024. "Artificial intelligence to automate the systematic review of scientific literature." arXiv:2401.10917. https://arxiv.org/abs/2401.10917

## Evidence Maps And Living Reviews

Evidence maps and living reviews make a field inspectable over time. They help show where evidence exists, where gaps remain, and where updates change conclusions.

Closest overlap:

- matrix-based coverage
- gap identification
- update cadence
- visible source structure

Key difference:

- this framework centers conflict and competing causal theories
- the map is not only a literature inventory; it is a disagreement engine

Representative references:

- Elliott et al., 2014. "Living Systematic Reviews: An Emerging Opportunity to Narrow the Evidence-Practice Gap." PLOS Medicine. https://journals.plos.org/plosmedicine/article?id=10.1371/journal.pmed.1001603
- Mallavarapu et al., 2026. "Interactive Evidence Maps for Visualizing and Understanding Systematic Reviews." arXiv:2603.28802. https://arxiv.org/abs/2603.28802

## Argument Mining And Argument Technology

Argument mining extracts claims, premises, supports, attacks, and argumentative structures from text.

Closest overlap:

- claim structure
- support and undercut relationships
- argument maps
- debate and disagreement representation

Key difference:

- this repo starts from evidence cards and study methods, not only argument text
- the target is not persuasive argument quality; it is evidence-grounded theory comparison

Representative reference:

- Li et al., 2025. "Large Language Models in Argument Mining: A Survey." arXiv:2506.16383. https://arxiv.org/abs/2506.16383

## Literature-Based Discovery

Literature-based discovery uses published knowledge to generate hypotheses, often by linking concepts that have not been connected directly.

Closest overlap:

- hypothesis generation from literature structure
- cross-source connection
- next-question discovery

Key difference:

- this framework does not claim discovery
- generated questions are ranked by their ability to discriminate between existing theories
- unresolved conflict matters more than novel association

Representative references:

- Swanson, 1986. "Fish oil, Raynaud's syndrome, and undiscovered public knowledge." Perspectives in Biology and Medicine.
- Smalheiser and Swanson, 1998. "Using Arrowsmith: a computer-assisted approach to formulating and assessing scientific hypotheses." Computer Methods and Programs in Biomedicine.

## Scientific Claim Verification

Scientific claim verification binds claims to supporting or refuting evidence.

Closest overlap:

- claim-evidence binding
- retrieval of relevant scientific evidence
- support/refute/insufficient categories
- rationale extraction

Key difference:

- this framework handles multi-theory disagreement rather than one claim at a time
- a finding can support, undercut, or scope-limit multiple theories
- unresolved conflict is preserved instead of forced into a simple label

Representative references:

- Wadden et al., 2020. "Fact or Fiction: Verifying Scientific Claims." arXiv:2004.14974. https://arxiv.org/abs/2004.14974
- Wadden et al., 2022. "SciFact-Open: Towards open-domain scientific claim verification." arXiv:2210.13777. https://arxiv.org/abs/2210.13777

## Novelty Boundary

Reasonable claim:

This project combines evidence-carding, conflict autopsy, competing-theory mapping, diagnostic scoring, adversarial review, and question generation into a manual-first autoresearch loop for domains without clean scoreboards.

Overclaim to avoid:

This project should not claim to invent evidence synthesis, automate scientific discovery, replace systematic review, solve claim verification, or prove truth in contested fields.
