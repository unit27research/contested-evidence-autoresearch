# Carverton Transit Example

Carverton is a fictional city transit system. The example exists to show the loop without touching biomedical, legal, safety-critical, or live policy claims.

## Contested Field

Carverton has run five major transit reforms in twelve years. On-time performance, ridership, and cost recovery remain poor. Four theories compete:

- T1 Schedule Fragility
- T2 Maintenance Debt
- T3 Governance Churn
- T4 Demand Mismatch

The example shows one cycle:

1. evidence cards
2. theory map
3. study autopsies
4. question backlog
5. cycle report

The point is not to solve Carverton. Carverton is not real. The point is to show how the framework preserves conflict and converts it into better next questions.
