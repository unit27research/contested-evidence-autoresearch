# LLM Benchmark Reliability Pilot

This is a small real-source pilot for Contested Evidence Autoresearch.

The pilot question:

> What kind of claim can an LLM benchmark score responsibly support?

This example does not evaluate any specific model. It does not decide whether a benchmark is good or bad. It uses a few public sources to show how the field kit handles a contested evaluation question.

## Why This Pilot Fits

LLM benchmark interpretation is a good first public pilot because:

- it is close to autoresearch and AI evaluation
- sources are public and inspectable
- the disagreement is about constructs, scope, contamination, and metric interpretation
- the output can be a narrower claim and a better question, not a final verdict
